What gets exported

Each click exports all 7 Sisense analytics datasets from your CloudShare BI instance. You get a single ZIP file (cloudshare_analytics_YYYY-MM-DD.zip) with one CSV per dataset, up to 50,000 rows each.

cost_control.csv

Cost Control

Use case, project, course, policy, both cost variants (lowercase + uppercase), usage dates.

user_hours_utilization.csv

User Hour Utilization

Course name, user hours consumed, user email, usage date, class type, user role, project.

experiences_participation.csv

Experiences & Participation

Experience names, types, hashed IDs, country, max participation counts, usage dates.

attendance_completion.csv

Attendance & Completion

Time/drop-off buckets, participant names & hashed IDs, user hours, attendance flag.

blueprints_environments.csv

Blueprints & Environments

Env names, blueprints, owners, disk GBH, RAM GBH, runtime & inactive hours, create dates.

resource_utilization.csv

Resource Utilization

Project & env name, disk GBH, memory GBH, source type, region, date.

project_members_activity.csv

Project Members Activity

Member names, emails, roles per project ? experience created/instructed counts, invitations, participant count, user hours, completion rate, environments owned.

Common questions
Does this send data to an external server?
No. The bookmarklet runs entirely in your browser tab on bi.cloudshare.com. All queries go directly to Sisense's internal JAQL API using your existing session. The ZIP is built locally and downloaded to your machine.
What if I click it on the wrong page?
If you're not on bi.cloudshare.com, a confirm dialog appears asking if you want to navigate there. Click OK and it redirects you to the CloudShare Analytics dashboard automatically.
What if a dataset fails?
Failed datasets are marked red in the overlay and skipped. The export continues with the remaining datasets, and you'll still get a ZIP with everything that succeeded. No failure blocks the rest.
How many rows does it pull?
Up to 50,000 rows per dataset. For most CloudShare accounts this covers all data. If a cube is larger, Sisense returns the first 50k rows.
Do I need to install anything?
No. It's a browser bookmark. Drag it once, use it as many times as you want. Works in Chrome, Edge, and Firefox.
How long does it take?
Usually 15?45 seconds total. Datasets run sequentially to avoid hammering Sisense. The overlay shows each dataset's status in real time as it completes.
