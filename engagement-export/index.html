How it works

This tool runs entirely in your browser using your existing CloudShare session. It reads engagement data from CloudShare's APIs and generates downloadable files locally. No data is sent to any external server. The export script is stored in your browser's localStorage after setup.

Version 10 • Works with Chrome, Edge, Firefox • Requires an active CloudShare login
